Why so much better?
Radiation resistance scales as D4. A 3× larger diameter yields 81× more Rrad, while conductor loss only increases 1.3×. Thicker tube cuts loss a further 2.3×.

Engineering Formulae

L = μ₀ · b · [ln(8b/a) − 2]

C = 1 / (ω² · L)

Rr = 320π4(A / λ²)²

δ = 1 / √(π f μ₀ σ)

Rloss = Cloop / (π d σ δ)

η = Rr / (Rr + Rloss)

Q = ωL / (Rr + Rloss)

Vcap = √(P/Rtot) · ωL
